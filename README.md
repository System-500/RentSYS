# RentSYS - Car Rental Management System

A comprehensive Car Rental Management System designed for efficient fleet and rental management. This project is a CRUD application built with a distributed architecture, ensuring robust logic separation and data integrity.

## 🛠 Technologies

* **Frontend:** React, CSS
* **Backend:** Node.js, Express.js 
* **Database:** MongoDB (using 4 collections) 

##  Key Features

* **Distributed Architecture:** The backend acts as a REST API server, handling business logic and JSON data communication.
* **Role-Based Access Control:** Hierarchical system for Administrators and regular Users.
* **Security:** Implemented JWT (JSON Web Token) for secure authentication and authorization.
* **Full CRUD Functionality:** Complete cycle of creating, reading, updating, and deleting records for vehicles and rentals.
